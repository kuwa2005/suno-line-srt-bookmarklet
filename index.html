Suno 行単位 SRT ブックマークレット

Suno の /song/ または /edit/ ページから、 タイミング付き歌詞 API を使って SRT をダウンロードします。 [Intro] などの [...] タグ行は除去し、歌詞 1 行 = SRT 1 ブロックです。

GitHub リポジトリ

このボタンをブックマークバーへドラッグ

Suno 行SRT 取得

ドラッグできない場合は、下の「手動登録」を使ってください。

使い方

  1. Chrome で Suno にログインする
  2. 対象曲の /song/{id} または /edit/{id} を開く
  3. ブックマークバーの Suno 行SRT 取得 をクリック
  4. {曲ID}-lines.srt がダウンロードされ、行数が alert で表示されます

手動登録する場合

  1. ブックマークマネージャで新規ブックマークを作成
  2. 名前: Suno 行SRT 取得
